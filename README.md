# UB Student Exchange

## Overview
This is a Java console application that allows University of Botswana students to buy and sell items like textbooks, electronics, and clothing.

## How to Run

Compile:
javac -d bin src/app/*.java src/model/*.java src/service/*.java src/contracts/*.java src/util/*.java src/exceptions/*.java

OR 
javac -d bin src/**/*.java

Run:
java -cp bin app.Main

## Features
- Add a listing
- View all listings
- Search listings
- Remove listings
- Market summary report
